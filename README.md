# CODE Network

A simple web app for managing contacts in the CODE community.

## What it does

- Add contacts
- View all contacts
- Store contact info in a database

## How to run

```bash
cd Project.py
source venv/bin/activate
python3 app.py
```

Then go to: http://127.0.0.1:8080

## Technologies used

- Python
- Flask
- SQLite
- HTML/CSS

## Files

- `app.py` - main application
- `views/` - HTML pages
- `public/css/` - styling
- `contacts.db` - database

That's it!
